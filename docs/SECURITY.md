# Security

## Principles

- Agent 只建立出站连接，不接受公网主动入站连接
- 设备注册使用短期注册码与一次性票据，不下发长期主密钥
- 高风险操作必须走登录态、权限校验、二次确认与 step-up
- 终端数据面和控制面分离，避免普通 API 中间件误处理双向字节流

## MVP controls

- 注册票据短时有效并可单次消费
- 会话建立前验证租户、设备可见性、会话权限
- 设备可被远程禁用，禁用后拒绝新会话并尽快清理旧会话
- 审计默认记录会话元数据与命令摘要，不默认开启全量留存

## Agent-side controls

- 本地记录安装、自启、升级、SSH 检查等关键事件
- 显式上报平台、架构、版本、能力，不伪装支持状态
- 平台不满足条件时返回可读诊断，不进入半安装态
